# Zerqis docs

**Problem:** Cross-repo dependencies are invisible until something breaks. **Solution:** Zerqis scans connected GitHub repos, builds a dependency graph, and surfaces impact on the graph, in the UI, and on **PR comments** (plus optional AI explanation). **Support:** Java/TS/Python/Ruby/Go-style services, GitHub + Slack, optional Anthropic.
